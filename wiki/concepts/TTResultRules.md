# tt-result 시스템 적용 규칙 V1.0

**우선순위: 0 (최고)** — 이 규칙은 시스템의 근간이며, 위반 시 다른 작업을 중단하고 수정한다.

---

## 규칙 1. 엑셀 업로드 기반 대회 구성

모든 대회의 참가선수 명단과 대진표 등을 **엑셀로 업로드하여 처리**할 수 있어야 한다.

- **현재 구현 상태**: 52nd에서 엑셀 업로드 구현 완료, 대학연맹에도 적용됨
- **확인 필요**: 모든 대회 유형(개인전/단체전/복식/혼합복식)에서 엑셀 업로드가 동작하는지

---

## 규칙 2. 예선리그 → 본선 토너먼트 연계

예선리그의 각 조 1, 2위가 추첨배정에 들어와 **대진표 번호를 입력 받음과 동시에 대진표 작성을 구성**할 수 있도록 한다.

- 예선 결과 → 본선 진출자 자동 선정 → 추첨 번호 입력 → 대진표 자동 생성
- **현재 구현 상태**: 52nd에서 조별 1,2위 → 본선 대진표 생성 구현됨 (draw.py)
- **확인 필요**: 추첨 번호 입력 UI가 자연스러운지, 대진표 즉시 생성되는지

---

## 규칙 3. 공방률 (동률 처리) — 핵심 규칙

예선리그에 적용되는 공방률 규칙:

### 3-1. 4인 이상 리그전 동률 발생 시
- 게임의 승/패와 승점에 따라 동률이 발생할 수 있다

### 3-2. 2명 동률 → 승자승 원칙
- 4인 이상에서 2명이 동률인 경우: **맞대결(승자승)** 결과로 결정

### 3-3. 3명 동률 → 2단계 해결
동률인 선수들만의 성적으로 결정:

> **1단계**: 3명 간 경기의 **게임 득/실**로 계산
> **2단계**: 게임 득/실이 동률이면 → **점수 합산 득/실**로 결정

- **현재 구현 상태**: tiebreak.py에서 ITTF 기반 공방률 구현됨
- **검증 필요**: 3명 동률 시 1단계(게임 득/실) → 2단계(점수 득/실) 순서가 정확한지

---

## 규칙 4. 국가대표 선발전 — 리그전 특수 규칙

### 4-1. 리그전 규모
- 적게는 **8인**부터 많게는 **16인**까지 단일 리그전 가능

### 4-2. 예선 → 본선 리그전 (성적 반영)
- 예선리그를 조별로 실시 → 각 조 1, 2등 선정 → **본선 리그전** 실행
- **예선전의 성적을 본선에 반영**하여 적용

### 4-3. 공방률 필수 적용
- 예선, 본선 순위 결정에서 **반드시 공방률 적용**
- 동률이 곳곳에서 발생되기 때문

- **현재 미구현**: 16인 리그전 UI, 예선 성적 본선 반영 로직
- **확인 필요**: 현재 tiebreak.py가 16인 리그에서도 정상 동작하는지

---

## 규칙 5. 단체전 — 예선리그 + 본선 토너먼트

### 5-1. 경기 방식 유연성
- 4단1복이 아닌 **5단식**으로도 진행 가능
- 연맹별·대회별로 경기 방식이 다를 수 있음

### 5-2. 단체전 UI 요구사항
- 예선리그 형식으로 표시하되, **팀 이름을 클릭하면 입력 모달** 표시
- 모달에서 제공해야 할 기능:
  1. **오더용지 확정** (출전 선수 순서)
  2. **기록지 인쇄**
  3. **결과 수정**

- **현재 구현 상태**: 52nd에서 7경기(6단식+1복식) 구현, 대학연맹 확장 중
- **확인 필요**: 5단식 전용 모드, 모달 UI에서 3가지 기능 모두 제공되는지

---

## 규칙 6. 기록지 및 대진표 출력 규격

### 6-1. 기록지
- 모든 기록지는 **A4 세로 배열 기준 1장을 초과하지 않아야** 한다

### 6-2. 대진표
- 256강 대진표의 경우 **4장으로 분할 인쇄** 가능해야 한다
- (화면 표시는 단일 뷰, 인쇄 시 4장 분할)

- **현재 구현 상태**: 기록지 A4 1장 구현됨 (ScoreSheet), 인쇄 CSS 적용
- **확인 필요**: 256강 대진표 4장 분할 인쇄 기능 미구현

---

## 규칙 7. 기권자 표시 — 취소선 처리

- 기권자는 참가신청에서 **삭제가 아닌 취소선** 처리
- 참가는 했지만 기권했다는 것을 명확히 표시
- CSS: `text-decoration: line-through`

- **현재 구현 상태**: 대학연맹에서 기권 취소선 구현됨 (2026-04-16)
- **확인 필요**: 모든 화면(명단, 대진표, 결과)에서 일관된 취소선 적용
