# 2026-04-13: 대학연맹 시스템 완성 — 전 종목 대진표 + 검사결과 + 일정

## 주요 작업 (후반부)

### 1. 대진표 자동 생성 (bracket_generator.py)
- generate_bracket(): draw_results → tournament_matches 자동 생성 + BYE 자동 진출
- generate_team_bracket(): 단체전 전용 (team_draw → team_matches)
- 16강/32강/64강 지원, 모든 라운드 빈 매치 미리 생성
- _place_in_next_round(): 승자 다음 라운드 자동 배치
- _place_team_next(): 단체전 승자 자동 배치

### 2. 전 종목 대진표 완성
- 남자복식 32강 (30조 + 2BYE) — 엑셀 업로드 → 대진표 자동 생성
- 여자복식 32강 (24조 + 8BYE) — 엑셀 업로드 → 대진표 자동 생성
- 혼합복식 64강 (53조 + 11BYE) — 엑셀 업로드 → 대진표 자동 생성
- 남자단체전 16강 (10팀 + 6BYE) — 수동 대진번호 배정 → 대진표 자동 생성
- 64강 시드 매핑(SEED_TO_DRAW_64) 추가

### 3. 대진번호 배정 UI
- 단체전: "팀 목록 불러오기" → 등록된 팀에 대진번호 입력 → 대진표 생성
- 개인전: "진출자 목록 불러오기" → 조별 1,2등에 대진번호 입력 → 대진표 생성
- 빈 번호 자동 BYE 채움

### 4. 경기결과 입력 모달
- 복식/혼합 대진표에서 매치 클릭 → 5세트 점수 입력 모달
- POST /match/{cat}/{round}/{num} → 결과 저장 + 승자 자동 진출
- loadGenericBracket(): 공통 bracket 렌더링 함수 (복식/혼합 공유)

### 5. 기록지 인쇄
- 라운드별 기록지 버튼 (64강/32강/16강/8강/준결승/결승)
- BYE 매치 제외, 실제 대결만 출력
- DOUBLES SCORE SHEET / MIXED DOUBLES SCORE SHEET 자동 표기
- 기록지 A4 1장 맞춤 (높이 축소)

### 6. 검사결과 (선수 검증)
- 참가선수(DB) vs KTTA 선수명부(엑셀) 크로스체크

- 김동한→김동환 오타 발견 및 수정 (검사의 실효성 입증)
